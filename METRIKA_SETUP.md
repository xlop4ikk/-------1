# Настройка воронки продаж в Яндекс Метрике

## Цели для настройки

Для отслеживания воронки продаж необходимо создать 5 целей в Яндекс Метрике:

### Шаг 1: Просмотр товара/каталога (100%)
- **Тип цели:** JavaScript-событие
- **Идентификатор цели:** `ViewProduct`
- **Где фиксируется:** `catalog.html` (при загрузке страницы)
- **Описание:** Пользователь просматривает каталог товаров

### Шаг 2: Добавление в корзину (X%)
- **Тип цели:** JavaScript-событие
- **Идентификатор цели:** `AddToCart`
- **Где фиксируется:** `index.html`, `catalog.html` (при клике на кнопку "В корзину")
- **Описание:** Пользователь добавил товар в корзину
- **Параметры:** передаются name, price, quantity товара

### Шаг 3: Переход в корзину (Y%)
- **Тип цели:** JavaScript-событие
- **Идентификатор цели:** `ViewCart`
- **Где фиксируется:** `cart.html` (при загрузке страницы)
- **Описание:** Пользователь перешёл на страницу корзины для оформления заказа

### Шаг 4: Оформление заказа (Z%)
- **Тип цели:** JavaScript-событие
- **Идентификатор цели:** `Checkout`
- **Где фиксируется:** `cart.html` (при отправке формы оформления заказа)
- **Описание:** Пользователь начал процесс оформления заказа

### Шаг 5: Покупка (W%)
- **Тип цели:** JavaScript-событие
- **Идентификатор цели:** `Purchase`
- **Где фиксируется:** `thank-you.html` (при загрузке страницы подтверждения)
- **Описание:** Пользователь завершил покупку
- **Параметры:** передаются transaction_id, value, currency, items

---

## Как настроить цели в Яндекс Метрике

1. Откройте Яндекс Метрику → Выберите счётчик (106701813)
2. Перейдите в **Настройки** → **Цели**
3. Нажмите **Добавить цель**
4. Для каждого шага воронки:
   - Выберите тип условия: **JavaScript-событие**
   - Введите идентификатор цели (например, `ViewProduct`)
   - Сохраните цель

5. После создания всех целей настройте воронку:
   - В списке целей найдите цель `Purchase`
   - Нажмите **Настроить воронку**
   - Добавьте цели в порядке: `ViewProduct` → `AddToCart` → `ViewCart` → `Checkout` → `Purchase`
   - Сохраните воронку

## E-commerce данные

Сайт настроен на передачу данных в dataLayer для расширенной электронной коммерции:

### Добавление в корзину:
```javascript
window.dataLayer.push({
    'event': 'add_to_cart',
    'ecommerce': {
        'items': [{
            'item_name': 'Название товара',
            'price': 29999,
            'quantity': 1
        }]
    }
});
```

### Покупка:
```javascript
window.dataLayer.push({
    'event': 'purchase',
    'ecommerce': {
        'transaction_id': 'ORDER_1234567890',
        'value': 29999,
        'currency': 'RUB',
        'items': [{
            'item_name': 'Название товара',
            'price': 29999,
            'quantity': 1
        }]
    }
});
```

## Проверка работы

1. Откройте консоль браузера (F12)
2. Перейдите на `catalog.html`
3. В консоли выполните: `console.log(window.dataLayer)`
4. Добавьте товар в корзину
5. Проверьте, что в dataLayer появилось событие `add_to_cart`
6. Перейдите в корзину → оформите заказ
7. На странице благодарности проверьте событие `purchase`

Также можно использовать **Яндекс Метрика → Отчёты → Стандартные отчёты → Конверсии** для просмотра статистики по целям.
